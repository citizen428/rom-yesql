* [API docs](http://rubydoc.info/gems/rom-yesql)
* [ROM user docs](http://rom-rb.org/learn)

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rom-yesql'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install rom-yesql
